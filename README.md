# 🚗 Smart Parking Management System

A **Full Stack Web Application** that helps manage parking lots and parking slots efficiently.
This system allows administrators to create parking lots, manage slots, and monitor parking availability.

The project is built using **React, Node.js, Express.js, and MongoDB** following a modern **REST API architecture**.

---

# 📌 Features

* 🔐 User Authentication (Register & Login)
* 👤 User Management System
* 🅿️ Parking Lot Creation
* 🚘 Parking Slot Management
* 📊 Parking Availability Tracking
* 🔄 RESTful API Integration
* ☁️ Image Upload using Cloudinary
* 📬 API Testing with Postman
* 🎨 Modern Dashboard UI with React + Tailwind CSS
